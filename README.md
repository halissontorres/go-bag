## What is Gobag?

> **Gobag** is a modern Go library that provides strongly-typed data collections and first-class enum support, designed to bring safety, expressiveness, and performance to your applications.

> Inspired by Go’s simplicity and enhanced with powerful abstractions, Gobag treats enums and collections as first-class citizens, working seamlessly together.
